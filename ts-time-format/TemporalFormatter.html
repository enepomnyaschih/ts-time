Description#

T
Type to format.
Date/time-related type string formatter. The result equals to concatenation of write method call results on all components. You may specify components explicitly by using 'of' method of a respective subclass, or implicitly by using 'ofPattern' method of a respective subclass to parse a string pattern.

Constructor#
